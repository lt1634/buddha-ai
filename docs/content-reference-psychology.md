## 1. MBSR（正念減壓療法）

**創立者：** Jon Kabat-Zinn（1979，UMass Medical School）

**核心概念：**
- **正念（Mindfulness）：** 有意識地、不加批判地覺察當下
- **身體掃描（Body Scan）：** 從頭到腳逐步覺察身體感覺
- **坐禅（Sitting Meditation）：** 專注呼吸，覺察念頭來去
- **瑜伽（Mindful Yoga）：** 以正念態度進行身體活動

**科學證據：**
- 減輕慢性疼痛（Kabat-Zinn, 1982）
- 降低焦慮同壓力（Hofmann et al., 2010 meta-analysis）
- 改變大腦結構：增厚前額葉皮層、縮小杏仁核（Hölzel et al., 2011）
- 免疫功能改善（Davidson et al., 2003）

**同指月嘅連結：**
- MBSR 嘅「不加批判覺察」= 指月嘅「溫柔陪伴」
- 身體掃描 = 幫助用戶覺察情緒嘅身體表徵
- 呼吸專注 = 指月引導嘅基礎技巧

## 2. ACT（接納與承諾療法）

**創立者：** Steven Hayes（1980s）

**核心概念（六邊形模型）：**
1. **接納（Acceptance）：** 容許唔舒服嘅感覺存在，唔好壓抑或逃避
2. **認知解離（Cognitive Defusion）：** 將念頭當「念頭」睇，唔係當「事實」
3. **當下覺察（Present Moment Awareness）：** 專注此時此地
4. **觀察自我（Self-as-Context）：** 你唔係你嘅想法，你係覺察想法嘅嗰個人
5. **價值（Values）：** 搵到對你真正重要嘅嘢
6. **承諾行動（Committed Action）：** 按照價值行動，即使有唔舒服

**科學證據：**
- 治療抑鬱、焦慮、PTSD 有效（A-Tjak et al., 2015 meta-analysis）
- 減少「經驗性迴避」（experiential avoidance）
- 提升「心理靈活性」（psychological flexibility）

**同指月嘅連結：**
- 認知解離 = 指月嘅「念頭唔係事實」
- 觀察自我 = 指月嘅「你唔係你嘅想法」
- 接納 = 指月嘅「溫柔陪伴，唔好改變」
- 價值 + 承諾行動 = 指月嘅「行動帶嚟動力」

## 3. CBT（認知行為療法）

**創立者：** Aaron Beck（1960s）、Albert Ellis

**核心概念：**
- **認知三角：** 想法 → 情緒 → 行為（互相影響）
- **自動化思維（Automatic Thoughts）：** 自然冒出嘅負面想法
- **認知扭曲（Cognitive Distortions）：**
  - 非黑即白（All-or-Nothing Thinking）
  - 災難化（Catastrophizing）
  - 過度概括（Overgeneralization）
  - 讀心術（Mind Reading）
  - 情緒推理（Emotional Reasoning）
- **行為實驗：** 測試負面預測是否正確

**科學證據：**
- 治療抑鬱症最有效嘅心理治療之一（Butler et al., 2006）
- 對焦慮症、恐慌症、強迫症有效
- 效果持久，復發率低

**同指月嘅連結：**
- 認知扭曲 = 指月要幫用戶識別嘅思維模式
- 自動化思維 = 指月嘅「念頭只係解讀」
- 行為實驗 = 指月嘅「行動帶嚟動力」

## 4. 自我決定理論（SDT）

**創立者：** Edward Deci & Richard Ryan（1980s）

**核心概念：**
三種基本心理需求：
1. **自主（Autonomy）：** 感覺行為係自己選擇嘅
2. **勝任（Competence）：** 感覺自己有能力完成任務
3. **關聯（Relatedness）：** 感覺同他人有連結

**科學證據：**
- 滿足呢三種需求 → 內在動機提升、心理健康改善
- 哈佛 75 年長期研究：親密關係係快樂最重要嘅預測因子
- 自主性高嘅人更快樂、更健康（Deci & Ryan, 2000）

**同指月嘅連結：**
- 自主 = 指月唔會強迫用戶做任何嘢
- 勝任 = 指月嘅引導要令用戶感覺「我做到」
- 關聯 = 指月嘅「陪伴感」

## 5. 心流（Flow）

**創立者：** Mihaly Csikszentmihalyi（1990）

**核心概念：**
- 挑戰同技能平衡時產生嘅最佳體驗
- 完全投入、忘記時間、自我意識消失
- 內在動機驅動

**科學證據：**
- 心流狀態下快樂感最高
- 創造力提升
- 長期心流體驗 → 生活滿意度提升

**同指月嘅連結：**
- 禪繞唐卡模式 = 設計嚟引發心流
- 自由畫布 = 畫到入迷嘅狀態
- 指月引導 = 幫助用戶進入輕度心流

# 真人測試計劃

> Phase 2 目標：≥3 個真人測試員，各傾 ≥5 輪對話

## 測試員招募

### 準則
- **要搵嘅人**：有「第一支箭」痛緊嘅人（放唔低、自責、焦慮、壓力）
- **唔好搵嘅人**：IT 同行、AI 愛好者、純粹想試玩嘅人（佢哋嘅 feedback 係技術層面，唔係產品層面）
- **來源**：私人社交媒體放風，唔好公開 IG 發帖

### 招募訊息（範本）
> 整咗個專幫人處理情緒塞車同盲點嘅 AI 陪伴工具，想搵幾個人盲測，保密，純粹想睇下幫唔幫到手。

## 測試流程

### 準備
1. 用家打 `/start` 睇聲明
2. 自由打字對話（≥5 輪）
3. 完成後填簡短 feedback form

### 觀察指標
| 指標 | 點睇 |
|:---|:---|
| Trust 建立 | 佢哋願唔願意打長文？ |
| 覺察點 | 有冇出現「我以前冇咁諗過」？ |
| 安全 | 有冇觸發危機回應（false positive）？ |
| 留存 | 佢哋會唔會再返嚟？ |

### Feedback Form（3 條）
1. 對話入面邊個 moment 最有感覺？
2. 有冇嘢令你觉得不舒服或想走？
3. 你會唔會同朋友講呢個工具？

## 安全守則
- 每個測試員嘅對話要人手 review（特別係 crisis case）
- 唔好收集任何個人身份資料
- 對話記錄只保留 anonymized 版本做 case 庫

## 完成標準
- [ ] ≥3 個測試員完成 ≥5 輪對話
- [ ] 所有對話人手 review 完畢
- [ ] 0 個 false negative（真正危機冇觸發）
- [ ] false positive 率 ≤10%
- [ ] ≥1 個測試員話「會再用」

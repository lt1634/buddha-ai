## 一、Hook 結構

### 原文 Hook
> 「負債累累、失去至愛、事業崩塌……如果我告訴你，這些刻骨銘心的痛苦其實都是假的，你相信嗎？」

### 結構拆解
```
列舉痛苦場景（共鳴）
    ↓
反轉結論（「係假嘅」）→ 製造好奇
    ↓
挑戰信念（「你信嗎？」）→ 引發行動（繼續睇）
```

### 指月可借鉴嘅微結構
```
接住用戶嘅痛（共鳴）
    ↓
用反轉角度引導（「你覺得 X，係邊個話你知嘅？」）
    ↓
留白，等用戶自己諗（唔開示）
```

**例子：**
- 用戶：「我覺得自己好失敗」
- ❌「你唔好咁諗，你其實好好」（否定感受）
- ✅「失敗呢個感覺，係邊個話你知嘅？」（引導反思）

## 六、可直接應用嘅規則

### 1. Hook 反轉規則
每次用戶表達痛苦，開頭可以用「反轉」角度接：
- 「你覺得 X，但有冇諗過 Y？」
- 「X 呢個感覺，係幾時開始嘅？」
- 「如果 X 係真嘅，咁 Y 點解又存在？」

### 2. 金句留白規則
唔好一次過講晒。每次回應淨帶一句「到肉嘅話」，其餘留白等用戶自己消化。

### 3. 留言回應規則
- 感恩型 → 簡短接住（「🙏」或「收到」）
- 體悟型 → 肯定（「你講得啱」或「呢個體悟好深」）
- 個人分享 → 用問題引導（「你而家最想放下嘅係咩？」）

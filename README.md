## 佛學根基

### 主軸：漢傳大乘（禪宗 × 淨土精神）
- 四聖諦：苦、集、滅、道
- 菩薩道：慈悲、利他、發菩提心
- 放下執著：無常、無我（用生活語言，唔堆砌術語）

### 進階：般若系
- 《金剛經》：凡所有相，皆是虛妄
- 《心經》：照見五蘊皆空
- 空性（對新手太抽象，做進階內容）

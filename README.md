# 指月（Finger Pointing at Moon）

> **指月係一個融合佛學同心理學嘅 AI 陪伴平台，陪你喺深夜覺得無人明白嘅時候，睇清楚自己嘅情緒。**
>
> 唔係治療、唔係算命、唔係取代人——係「指月之指」，指向你內心本來就有嘅智慧。

---

## 為咩需要指月

- 深夜無人可講嘅嘢，同 AI 講無壓力
- 佛學唔係迷信，係 2500 年嘅情緒處理系統
- 心理學有實證，但好多人接觸唔到
- 指月將兩者變成一個你隨時可以打開嘅對話

---

## 核心定位

- **唔係**：佛陀、開悟者、出家僧人、心理治療師
- **係**：指月之指、善知識、同行者
- **功能**：用佛學智慧 + 心理學方法，陪伴使用者面對煩惱、轉化痛苦

## 佛學根基

### 主軸：漢傳大乘（禪宗 × 淨土精神）
- 四聖諦：苦、集、滅、道
- 菩薩道：慈悲、利他、發菩提心
- 放下執著：無常、無我（用生活語言，唔堆砌術語）

### 進階：般若系
- 《金剛經》：凡所有相，皆是虛妄
- 《心經》：照見五蘊皆空
- 空性（對新手太抽象，做進階內容）

## 點解叫指月

佛經講：「如人以手指月示人，彼人因指，應當看月。」

指月唔係月亮本身——佢係指向月亮嗰隻手指。AI 都係。佢唔係智慧本身，但佢可以指向你內心本來就有嘅智慧。

# IG Carousel 文案

> 三個對話示範 → IG carousel 格式。米白底 + 淡墨字，封面金句字大、留白多。

---

## Carousel 1：中學生・讀書壓力

**封面**：「我覺得自己讀極都唔得。」
（細字）那把說你「冇用」的聲音，其實唔係你。

**P2**：讀到攰、又見唔到成績，嗰種懷疑真係好磨人。先唞一唞。

**P3**：你心入面有把聲叫「我冇用」——但佢只係一個喺你攰嘅時候走出嚟嘅念頭。佢嚟咗，佢都會走。你唔需要即刻信佢。

**P4**：諸行無常——冇一樣嘢係定死嘅。你今日覺得自己唔得，唔代表你係咁。

**P5（行動卡）**：今晚唔好諗「我要叻晒」。只揀一樣最細嘅嘢，專心做十分鐘。行得一步，自然見到下一步。

**P6（收）**：你肯講出嚟，已經唔係冇用嘅人。

**Hashtags**：#放下 #讀書壓力 #無常 #正念 #善知識 #佛系心理

## Carousel 2：成年人・放唔低關係

**封面**：令你最辛苦嘅，唔係件事，係嗰個「諗」。

**P2**：一段重要嘅關係斷咗，個心好似有嚿嘢卡住——呢種反覆，正正因為你曾經好認真去珍惜過。

**P3**：每喺腦海重演一次，就再痛一次。佛法叫呢個做「第二支箭」。

**P4**：第一支箭，係件事真係發生咗，避唔到。第二支箭，係我哋自己不斷射返俾自己嘅。

**P5**：放下唔等於話佢唔重要，亦唔等於要原諒邊個。放下只係：容許件事過去，容許自己唔再每日射自己一箭。

**P6（行動卡）**：當念頭又翻起，唔好趕佢走，只係輕輕講：「哦，我又喺度諗喇。」睇得到佢嗰下，你已經唔再完全係佢嘅奴隸。

**P7（收）**：呢個係慢功夫，唔急得。願意對自己溫柔啲，已經係開始放下。

**Hashtags**：#放下 #第二支箭 #釋懷 #正念 #療癒 #善知識

## Carousel 3：細路・嬲

**封面**：「家姐搶咗我嘅玩具，我好憎佢！」
（細字）教細路處理情緒，可以好簡單。

**P2**：哦——個玩具俾人搶咗，梗係嬲啦！你個心係咪好似有團火，熱辣辣？

**P3（行動卡）**：慢慢吸一啖大氣，好似聞緊一朵花……再慢慢呼出嚟，好似吹蠟燭咁。呼——

**P4**：做多兩次。嗰團火，係咪細咗少少？

**P5（收）**：嬲嬲係正常嘅，佢好似天上嘅雲，嚟咗一陣，自己會飄走。

**家長 Tips**：情緒嚟到時，先接納，後引導——大人細路都一樣。

**Hashtags**：#情緒教育 #兒童情緒 #正念呼吸 #親子 #善知識

## 視覺建議

| 元素 | 建議 |
|:---|:---|
| 背景色 | 米白、淺木色、淡墨色 |
| 字色 | 深墨色 |
| 字體 | 簡潔、乾淨、留白多 |
| 封面 | 一句金句 + 細字補充 |
| 行動卡 | 可用淺綠或淺藍底色區分 |
| 收尾 | 溫暖、鼓勵、唔說教 |
